Instruções
Você recebeu a tarefa de desenvolver um sistema para gestão da imobiliária da família. Foi solicitado as seguintes funcionalidades:

Cadastrar os imóveis com as suas respectivas características
Cadastrar os clientes
Cadastrar o contrato de aluguel (valor, data de início, fim, etc)
Além disso, o sistema deve gerar alguns relatórios:

Listar os imóveis disponíveis para alguel
Listar contratos ativos
Clientes com mais contratos
Contratos expirando nos próximos 30 dias
O sistema deverá persistir todos os dados em um banco de dados.
O sistema deverá ser operado via terminal.

Entrega:
Adicione o link do seu repositório público no github.
O repositório deverá conter o projeto java, diagrama de classe da UML e o diagrama MER.
Os diagramas de classe devem estar no formato PDF ou JPG.

Arquivo SQL com as instruções de criação de tabelas e inserção de registros.

Dica: Adicione as imagens e informações extras no arquivo README.md da raiz do repositório.

Exemplos: https://github.com/leandersonandre/poo_trabalho_jdbc/

 

Avaliação:
Organização, implementação e corretude do código.
Dar um play e o programar executar corretamente.

Diagrama de classe refletindo o código desenvolvido

O diagrama MER refletindo as tabelas utilizadas no projeto.

Arquivo SQL com os scripts necessários.
